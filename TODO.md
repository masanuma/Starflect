## 🌟 プロダクト方針：毎日会いにいく「星の相棒」（**v0.1 本番稼働中**）
> 背景：数日使った実感として「パッとしない／ときめかない／普通の占いより地味」。
> 原因は **一発診断で終わり毎日の理由がない・結果が“報告書”・長文で快感の型がない**こと。
> 方針を **「診断ツール」→「毎日会いにいく“星の相棒”」** に転換する（方向性A、採用決定）。

### 核となる差別化：相棒育成ループ（“当てもの”を超える）
普通の占いは答え合わせをしないので「ただの当てもの」で終わる。そこを閉じて**関係性**にする。
ただし **“的中の採点”にはしない**（大半が外れて信頼を損なう＆景表法の優良誤認リスク）。
→ **答え合わせ＝対話と記録**。相棒はドヤらず「昨日どうだった？教えてくれてありがとう」と受け止め、
明日の語りに織り込む。価値は“当てる”ではなく **“わかってくれてる”**。使うほど蓄積し、
**あなた専用に育つ＝乗り換えられない**（本格星読み×AI×キャラ人格＝うちだけの武器）。

### 確定した設計判断
- **相棒＝あなたのほしキャラ**。診断で出た16タイプのキャラ（例：静かな火山）が、静的な結果ではなく
  **人格を持って毎日語りかける“生きた相棒”に昇格**する。所有感とブランドの一貫性を両取り。
- **“たまごっち感”は採用、ただし義務・罰・死は排除**。サボっても責めない／死なない／減点しない。
  放置しても関係は途切れず、戻れば温かく迎える（「おかえり」）。育つのは一方向（貯まる・深まる、減らない）。
  ＝安心して離れられる育成。メンタルケア用途で罪悪感メカニクスは毒。「また会いたい」は残し「会わなきゃ」は消す。
- **振り返りはタップ主体で軽く**（`いいことあった😊 / 微妙😔 / ふつう` 等）。ひとこと日記は任意のオマケ。
  - 理由：**日記を毎日書ける人はもうAIに相談している**。面倒だと人間は続かない。軽さ最優先。
- **細かい当たり外れは読まない**。むしろ **継続性からの計測**を主役に
  （例：タップの蓄積から「最近、気分が晴れないみたい」を相棒が気づく＝トレンド検出）。
- 占いは**“予言”から“朝の問いかけ＋夜の受け止め”**へ。星は会話のきっかけ、**商品は関係性**。

### 信号の経済（入力設計）＝「毎日＋簡単フィードバック」でどこまで“されていく”か
ほぼ手間ゼロの信号から、どれだけ豊かに導出できるかの設計。相棒は“育てる”ではなく痕跡から“育っていく”。

| 信号 | 手間 | 導出されるもの |
|---|---|---|
| 出生図（初回1回） | ゼロ | 相棒の素の人格・表の顔。Day1から効く土台 |
| 訪問の時間帯・曜日 | ゼロ(自動) | 生活リズム／朝型夜型／週末の落差 |
| 開く頻度・間隔 | ゼロ(自動) | 波を拾う(罰にしない)。「久しぶりの日」を温かく迎える材料 |
| よく触るテーマ | ゼロ(自動) | いまの関心(恋愛ばかり開く＝頭がそこにある) |
| 気分タップ 😊😔ふつう | 1タップ | 気分トレンド・移動平均 |
| 2タップ目「何系？」 | +1(任意) | 気分×領域(仕事の日に😔が多い) |
| 一言への反応(刺さった/ピンとこない) | 1(任意) | 何が本人に響くか＝人物像のチューニング |
| 相棒の「1日1問」遊びの二択(スキップ可) | 1(任意) | 心理プロファイルを“質問感なく”蓄積 |

- 明示的に聞くのは**1日1問まで・遊びの二択・スキップ可**。アンケートでなく“相棒があなたに興味を持ってる”演出。
- **隠れた自分エンジン**：本当の魔法は「聞かずに計算で出す」。
  - 気分 × トランジット相関 →「土星が絡む日にしんどくなりやすいみたい」＝あなた専用に調律された占い（他社が出せない必殺技）。
  - **出生図(アーキタイプ＝建前) と 日々の行動(実測) のギャップ**＝隠れた一面の正体。「表の顔は社交的なのに、ひとりの日に元気が戻ってるね」。星が素材、ギャップが商品。
- 到達目安：near-zero の手間でも **2〜3週で「わかってくれてる」に到達可能**。Day1〜3は出生図頼みで温かく → Week2〜3で“見抜き”が発火。
- 注意：小サンプルはノイズ。相関は断定せず「〜かも」トーン厳守（信頼＆景表法）。

### 相棒の声・トーン
- **“気持ち悪い寄り添い”を消す**。空虚な肯定の乱発（おべっか）はしない。悩んでいる人に刺さるのは、肯定ではなく
  **やさしい指摘・正直なフィードバック**。最初は多少の寄り添いはOK、でも core は“やさしい本音”。
- **星が“優しい本音”の乗り物**：批判や注意を「私が言う」でなく「**今の星の運行がこう出てるから**」に載せる。
  角を立てず、おべっかにも説教にもならずに背中を押せる。三点セット＝**温かさ＋正直さ＋星の具体**。
- **軸はあくまで星座占い**。今のトランジットから具体に落とす：「こういう傾向だから、ここは注意、ここは思い切って大胆に」。
  エッセンスを忘れない。占いの役割＝**背中を少し押して自信をつける／そっと注意する**。断定予言・不安煽りはしない。
- **距離感は「昔からの友人」**。ブランクがあっても責めず重くせず、「最近どうしてた？」で自然に再開。
  親密さは実際の利用履歴に合わせて調整（会ってない期間はそれ相応の距離で、でも温かく）。

### 相棒の正体・語り口（7日間ペーパーシミュで確定）
- **正体＝あなたのほしキャラ本人**（例「静かな火山」）。**あなたと同じ星でできた“同族の相棒”**。
  「もう一人のあなた」宣言はしない（気持ち悪い）。外から観察する助言者にもしない（監視感で怖い）。
- **一人称「わたし」**。自分のほしキャラ名をセリフで毎回名乗らない（肩書き表示で担保）。
  型の話は所有格でなく **「わたしたち、こういう日ほど…」と同じ側から**（例：「ミオの静かな火山は」はNG＝関係が不明瞭）。
- **メタ宣言をしない**：「当てにいかない」「隣で見てる」等は口に出さず態度で示す（宣言は逆に監視・恐怖を生む）。
- **星は「星の巡り／今日の星」**と言う。「空の様子」＝天気予報っぽいのでNG。
- **名前呼びはオプション（デフォルトは「あなた」）**。獅子座の月等には効くが多用は重い。使うなら再会・特別な瞬間だけ。頻度は別途単体で検証。

### 未来と振り返りのバランス（＝占いの本分を忘れない）
- 占いの本分は**未来予測**。振り返り（答え合わせ）は関係を作る装置だが、**ウエイトはむしろ未来側**でよい。
- 役割分担：**朝カード＝“今日”の未来（追い風／注意）／夜タップ＝振り返り（短く・見抜きの燃料）／週末＝“翌週”の未来予測**。
- **週末は「今週こうだったね」で終わらせない**。翌週の運行から **よさそうなこと（追い風）＋注意すること** を必ず出す。
  振り返りは3行程度に圧縮し、尺は未来に割く。→ 「来週こういう追い風」が**次に開く動機（楽しみ）**になる。

### 1日ループ（たたき台）
- 朝：相棒が“今日の星”を読んで一言＋そっと注意/追い風（未来向き）。今日のカード（色・キーワード）。
- 夜：「今日はどうだった？」を軽タップ（😊/😔/ふつう ＋任意で「何系？」）→ 相棒が受け止め、記憶に刻む。
- 週末：**圧縮した振り返り → 翌週フォーキャスト（追い風＋注意）**。未来にウエイト。
- 蓄積 → 仲良し度、語り口が親密に、過去を参照。“見抜き”は毎週やらず**たまに・不意に**（驚きの持続）。

### 検証済み：7日間ペーパーシミュ（架空ペルソナ「静かな火山」）
- Day1出会い→Day4に1問の伏線→Day7で回収、の弧が効くと確認。**Day4伏線→Day7見抜き**が“見られてた驚き”の主役。
- ②“見られてた驚き”と③“覚えててくれる関係”は蓄積前提＝**Week1後半で本領**。
- 最初の関門＝**Day1〜3をどう飽きさせず持たせるか**（朝カードの報酬＋出生図由来の「あなただから」の具体で持たせる）。
- 注意点：Day7の見抜きは長文になりがち→短く。見抜きは頻発させない。

### 実装の勘所（既知で解ける）
- **記憶の圧縮**：全履歴は毎回渡せない。定期的にAIが「この人プロフィール」を1枚に要約して保持（長期記憶チャットの定番）。
- **コスト**：日常の一言は安いモデル/テンプレ、要所だけ上位モデル。→ 「AIコスト対策」と直結。
- **データ永続化／引き継ぎ**：基本はローカルで育成（プライバシー＆軽さ）。機種変対策は
  **LINE風の「いったんクラウドへアップ → 新端末でダウンロードして復元」**方式（引き継ぎコード等）。
  フルログインを必須にせず“安心感”を出す。将来サブスク化のタイミングで正式アカウントに発展。
- **プライバシー**：良かった/嫌だったの記録＝センシティブな生活データ。当面ローカル完結を徹底、同期するなら同意と保護必須。

### 次プランで詰める設計論点
- **相棒との“最初の出会い”**（Day1の「今日から一緒に星を見ていくね」という契約シーン）の設計。ここが弱いと結果画面のままになる。
- **夜タップの粒度**：`😊 / 😔 / ふつう` だけか、**2タップ目で「何系？（仕事／恋愛／人間関係／なんとなく）」**を足すか。
  足すとトレンドが「最近“仕事”で😔が続いてるね」まで言える。手間とのトレードオフ。
- **気分トレンド検出**：タップ蓄積から「最近、気分が晴れないみたい」等を相棒が気づくロジック（移動平均・連続日数など）。
- **UXシミュレーションの進め方**：架空ペルソナで7日間を文言・タップ選択肢まで書き起こす（設計検証が速い）／
  本人テストプレイは深いが実時間で1週間かかる。まずは架空ペルソナ想定。

### ✅ MVP v0.1（A1＋B1／日常はテンプレ・AIゼロ）— 本番リリース済み `9735d58`
狙い：**毎日「朝の星＋夜タップ」ループでリテンション（また明日開く）が動くか**を最小コストで検証。

**実装した機能**
- **相棒ホーム**（2回目以降の起点）：罰なし挨拶（経過日数で出し分け）＋運勢＋相談室＋夜タップ＋週末まとめ。
- **ほしキャラが読む運勢**（`StarReading` 共有部品）：**今日/今週/今月・明日/来週/来月**を画面遷移なしで切替
  （来週=+7日、来月=翌月1日で `readFortune`）。**ラッキー/注意/注目バッジ**（アイコン＋文字）、**今日のラッキーカラー**（色名つき）。
- **夜の振り返りタップ**：😊😐😔＋任意で「何系？」→ テンプレで受け止め。同日は再質問しない。
- **週末まとめ＋翌週フォーキャスト**：直近7日の集計（簡易“見抜き”）＋来週の追い風/注意。土日表示。
- **データ**：`starflect-companion:v1 = { chart, starType, createdAt, lastVisitAt, daily{ 'YYYY-MM-DD':{ mood, domain?, forecastSeen } } }`（ローカルのみ）
- **GA4**：`companion_open` / `tap_mood` / `tap_domain` / `weekend_view`

**実装中に確定した追加の設計判断（重要）**
- **AIは「ほしキャラ相談室」に一本化**。別人格「AI占星術師」（ソロのAI鑑定・`/api/ai-reading`）を**廃止**。
  相談相手＝**あなたのほしキャラ本人**（システムプロンプトを本人設定に。一人称「わたし」・丁寧口調・「AI/占い師」と名乗らない）。
  → 表層＝ほしキャラが読む運勢（無料・テンプレ・一瞬）／深層＝同じほしキャラと相談室で会話（AI）の2層。
- **キャラは1つ＝あなたのほしキャラ**。「相棒」という別呼称もUI文言から排除して「ほしキャラ」に統一。
- **迎え入れボタンは廃止＝自動で相棒になる**。診断した時点で保存し、次回から相棒ホームが起点（「毎日そばに」を挙動で体現）。
- **診断は初回のみ**：データ無→診断カード／データ有→「あなたのほしキャラ「◯◯」と会話する」＋小さく「自分の情報を変更する」
  （フォームは `starflect-input` で前回値プリフィル済み）。
- **導線の出口/入口を両方確保**：相棒ホーム上部に「メインメニューへ」、ホームに相棒への入口。
- **表記・書式ルール**（今後もこれに従う）
  - ほしキャラ名は**必ず言語別の括弧**で囲む（JA「」/ EN "" / ES・FR・IT・PT «» / KO「」）。`quoted()` で一元化。
  - カードは**共通ヘッダー** `card-head`（ほしキャラのマスコット＋グラデタイトル18px/800＋説明）。
  - **説明文はセンタリングしない（左揃え）**。中央寄せは短い見出し・ラベル・設問のみ。
  - **カード間の余白は 16px 固定**（全カード `margin: 0 0 16px`）。
  - 見出し＝名詞句／ユーザーに回答を求める所だけ問いかけ（案C）。口調は**丁寧だが硬すぎない**で全画面統一。

- [ ] 広告は最後の手段（世界観を壊しやすい）
