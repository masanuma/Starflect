# デザイン刷新コンセプトメモ：ChatGPT感からの脱却

## 1. 「ChatGPT感」の原因分析
現在のUIがなぜAI（ChatGPT）っぽく見えるのか：
*   **均一なカードレイアウト**: すべてが同じ角丸・同じ影の白いボックスに入っている。
*   **標準的なグラデーション**: インターネット上でよく見る青紫系のグラデーション。
*   **情報の羅列**: AIが出力したテキストをそのまま枠に流し込んだような構成。
*   **動的体験の不足**: ページを開いた瞬間にすべての枠が用意されており、「占い」という神秘的な体験が静的。

## 2. Starflect v2 のデザイン方針
「便利ツール」ではなく「運命に触れる体験」へとシフトする。

### 2.1 配色：宇宙の深み
*   **背景**: 単なるグラデーションではなく、わずかにノイズが混ざったような深い紺色、あるいは星雲のような「深み」のあるテクスチャ。
*   **文字色**: 純粋な白 (#ffffff) ではなく、少し青みや黄色みが入った柔らかい色 (#e2e8f0, #fef3c7) を使い、目に優しく、かつ高級感を出す。

### 2.2 形状：オーガニック & フローティング
*   **境界線の排除**: 白いボックスを並べるのをやめ、背景にぼかしを入れた透過パネル（Glassmorphism）を使用。
*   **非対称性**: すべてをきっちり並べず、あえて少しずらしたり、天体アイコンが枠からはみ出したりするような、動きのあるレイアウト。

### 2.3 タイポグラフィ：物語性
*   **メイン見出し**: `Cinzel` や `Playfair Display` のような、伝統と品格を感じさせるセリフ体。
*   **行間と文字間**: テキストが詰まりすぎないよう、ゆとりを持たせたカーニング。

### 2.4 インタラクション：生命感
*   **スクロールアニメーション**: ユーザーが下にスクロールすると、星屑の中から文字が浮かび上がってくるような演出。
*   **ボタン**: 押した瞬間に星が弾けるようなエフェクト、あるいは光が走るようなアニメーション。

## 3. 具体的な改善イメージ（Before/After）
*   **Before**: 「太陽星座：獅子座」という見出しの白いカード。
*   **After**: 獅子座のシンボルマークが金色に輝き、その周りに「あなたを照らす生命の源」という情緒的なコピーが寄り添う、背景と一体化したセクション。
