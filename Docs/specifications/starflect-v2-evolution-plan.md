# Starflect v2 進化計画：次世代占星術体験の構築

## 1. プロジェクト概要
Starflectの技術的完成度を土台に、「AI感」を払拭した情緒的デザインと、ユーザーの定着・拡散を促す新機能を導入し、アプリのブランド価値と利用者数を最大化する。

## 2. デザインの抜本的見直し（Design Overhaul）
現在の「ChatGPT的」な標準的UIから、神秘的で没入感のある「占星術ブランド」へ脱却する。

### 2.1 ビジュアルコンセプト： "Deep Cosmos & Ethereal Light"
*   **カラーパレットの刷新**:
    *   標準的な紫・ピンクのグラデーションを廃止。
    *   深い真夜中の青 (#0a0b1e)、星雲の輝き (#1a1c3d)、柔らかい月光 (#e2e8f0)、そしてアクセントに黄金色 (#d4af37) を採用。
*   **タイポグラフィ**:
    *   見出しには優雅で神秘的なセリフ体（またはそれに準ずるWebフォント）を採用し、本文は可読性の高いサンセリフ体を維持。
*   **UI要素の形状**:
    *   角丸の白いボックスの連続を避け、背景と馴染む半透明（グラスモーフィズム）や、境界線をあえて曖昧にしたフローティング要素を活用。
*   **アニメーション（没入感）**:
    *   占い結果の表示時に、星座が浮かび上がるようなフェードイン、微細な星の瞬き（Particle Effect）を背景に導入。

### 2.2 コンポーネント別の改善点
*   **InputForm**: 入力画面を「作業」ではなく「儀式」のように演出。WheelPickerの背景に天球図を配置。
*   **StepByStepResult**: 情報量を整理し、カード形式を廃止。スクロールに合わせてストーリーが展開するようなレイアウト。
*   **AI Chat**: チャットバブル形式から、対話相手（AI占星術師）の存在を感じさせるインターフェースへ。

## 3. 新機能（集客・定着）
### 3.1 SNSシェアカード（Viral Expansion）
*   **機能**: 占い結果の要約（太陽・月・上昇星座のアイコンと一言）を、美しくデザインされた画像として生成またはCanvas描画。
*   **拡散**: ワンタップでX/Instagramへシェア。ハッシュタグ「#Starflect」を自動付与。

### 3.2 今日のラッキーアクション（Daily Retention）
*   **機能**: ユーザーの星座に基づいた「今日の一言アドバイス」をトップページに小さく表示。
*   **更新**: 毎日0時にAIが生成（またはキャッシュされたデータを配信）。

### 3.3 AIコンシェルジュ（Personalization）
*   **機能**: 文脈保持型チャットの強化。過去の占い結果や相談内容をふまえた、よりパーソナライズされた回答。

## 4. 技術的改善（Technical Excellence）
*   **StepByStepResult.tsx の分割**:
    *   5,000行の巨大ファイルを `FortuneHeader`, `ZodiacAnalysis`, `PlanetDetails`, `DailyAdvice` 等のコンポーネントに分割。
*   **Level2コードの物理削除**: 未使用のLevel2関連コードを完全に除去し、バンドルサイズを軽量化。

### Phase 3: AI対話体験の深化（2週間〜）
*   AIコンシェルジュの記憶・文脈保持機能の実装。
*   全体的なアニメーションのブラッシュアップ。
